# Corehubb OJT Project G2

##  Overview
Corehubb OJT Project G2 is a training project focused on building a Laptop Sales Management System.  
It includes backend, frontend, and database components, designed to simulate a real-world enterprise application.

##  Tech Stack
- **Backend**: Java Spring Boot
- **Frontend**: ReactJS
- **Database**: Microsoft SQL Server
- **Project Management**: Jira

##  Initiate DB 
Prepare the database
Create an empty SQL Server project.
Run Database_SQLserver_demo.sql inside that project to set up schema and sample data.
Start backend
Run the Spring Boot application.
